
# Client

## Composition
* ReactJS 15
* React Router 4
* Webpack 3
* Mobx

## Realtime Integration
* Horizon
* RethinkDB


## To Develop on Project

Run the following (in order):

1) To Start Rethinkdb

    `rethinkdb`

2) To Run Horizon Server

    `hz serve --dev`

3) To Start Client

    `npm start`
